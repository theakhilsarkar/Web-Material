# Tailwind CSS & React – Website Recreation Project

### Project Duration

72 Hours

### Total Marks

50

### Objective

Each student is required to develop a complete website using **React** and **Tailwind CSS**, recreating the assigned reference website with maximum accuracy in layout, styling, and responsiveness.

### Technology Requirements

* React (Latest Version)
* Tailwind CSS
* Deployment Hosting: Vercel or Netlify
* No external CSS frameworks or pre-built component libraries are allowed.
